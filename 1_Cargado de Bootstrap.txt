	Enrutamiento de bootstrap:
	__________________________


	_así, vamos al "head", y redireccionamos la ruta de css a la de nuestro proyecto, ej:


		Original:
		_________


		<!-- Bootstrap core CSS -->
    		<link href="../../dist/css/bootstrap.min.css" rel="stylesheet">



		Cambiarlo por:
		______________


		<!-- Bootstrap core CSS -->
   	 	<link rel="stylesheet" href="C:\desarrollos\bootstrap\css\bootstrap.min.css" rel="stylesheet">


	_así, tendremos re redireccionar todos las llamadas relacionales ha las rutas de nuestro proyecto
